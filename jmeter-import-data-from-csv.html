Intro

Today there is no change in source code ;) It'll be continuation of the last lesson about authorization. But you can use CSV files not only for authorization. Sometimes you need to send form with correct (different) data. You can also control your tests by files. There is no problem to read URL from file and send requests from jMeter. Believe me - you're free! I hope you know sth about CSV file format. But if you don't - it's so simple. New line is like a record. And the data (columns) are usually separated by commas. You can prepare data in Excel and save it as CSV file. It's simple and popular format.
